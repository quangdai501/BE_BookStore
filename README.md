## Run Locally

Clone the project

```bash
  git clone https://github.com/quangdai501/BE_BookStore.git
```

Go to the project directory

```bash
  cd BE_BookStore
```

Install dependencies

```bash
  yarn
```

Start the server

```bash
  yarn start
```
